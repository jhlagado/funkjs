# funk.js

A lightweight library of functional functions

```javascript
    pick(object, keys) 
    zipobject(props, values) 
    unzip(array) 
    argarray(args) 
    length(x) 
    assign() 
    map(x, f) 
    reduce(x, f, acc) 
    isarray(x) 
    isempty(x) 
    isobject(x) 
    isstring(x) 
    isboolean(x) 
    isfunction(x) 
    issymbol(obj) 
    filter(array, func) 
    all(x, f) 
    ispair(x) 
    cons(x, y) 
    isa(x, constructor) 
    existy(x) 
    startswith(string, target) 
    getkeys(a)
    isequal(a, b, maxdepth) 
```
